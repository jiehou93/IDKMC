# IDKMC
IDKMC--Irradiation and Desorption Kinetic Monte Carlo

本软件主要功能为模拟晶体材料中的缺陷及缺陷团簇的长时间演化。主要功能：
-高能离子辐照损伤的积累与演化（最多支持三束同时辐照）
-缺陷退火与回复
-低能氢/氦离子注入、渗透、热脱附
-多种缺陷钉扎下的元素扩散
......
